# 🎓 Student Helpdesk Chatbot

## 📌 Overview

This project is a chatbot built using IBM Watson Assistant. It helps students by answering common queries such as courses, class timings, and general information.

## 🚀 Features

* Greeting interaction
* Course information
* Class schedule details
* Exit conversation handling

## 🛠️ Technologies Used

* IBM Watson Assistant
* IBM Cloud
* JSON (chatbot export)

## 📁 Project Files

* chatbot.json (Exported chatbot from IBM Watson Assistant)
